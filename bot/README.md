# Meshtastic bot

1. requirements.txt - python requirements
2. meshtastic-bot.env expected in /etc/meshtastic-bot.env
3. meshtastic-bot.service expected in /etc/systemd/system/meshtastic-bot.service, **change 192.168.0.2 to your meshtastic IP**
4. meshtastic-bot.env expected in /opt/meshtastic-bot/app/meshtastic_bot.py
5. expected created user and group `meshtastic`
6. python virtual environment with installed requirements expected in /opt/meshtastic-bot/venv
